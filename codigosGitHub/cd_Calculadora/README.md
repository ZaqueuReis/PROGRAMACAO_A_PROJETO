Categorizacao de reais em ceduals

Este programa obtem um valor de entrada em reais e retorna a sua cassificao em cedulas de forma crescente, para exemplicar, imgine que ele recebeu como entrada um valor de 20 reiais, a saida esperada nesta siuacao é apenas 1 nota(s) de 20 $Reais.

## Funcionalidades

* Transacoes: sejam elas em lojas ou bancos, dado que o valor seja grande e dificil de se realizar a contagem manualmente

## Tecnologias utilizadas

* Linguagem: Python
* Outras ferramentas: Git, Vs Code
